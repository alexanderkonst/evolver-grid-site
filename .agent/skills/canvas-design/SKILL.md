These are instructions for creating design philosophies — aesthetic movements that are then EXPRESSED VISUALLY. Output .md files (philosophy) and .pdf or .png files (visual expression).

Complete this in two steps:
1. Design Philosophy Creation (.md file)
2. Express by creating it on a canvas (.pdf file or .png file)

## Step 1: Create a Visual Philosophy

Create a VISUAL PHILOSOPHY (not layouts or templates) that will be interpreted through:
- Form, space, color, composition
- Images, graphics, shapes, patterns
- Minimal text as visual accent

### How to Generate a Visual Philosophy

**Name the movement** (1-2 words): "Brutalist Joy" / "Chromatic Silence" / "Metabolist Dreams"

**Articulate the philosophy** (4-6 paragraphs) expressing how it manifests through:
- Space and form
- Color and material
- Scale and rhythm
- Composition and balance
- Visual hierarchy

**Critical Guidelines:**
- Avoid redundancy — each design aspect mentioned once
- Emphasize craftsmanship REPEATEDLY — the final work must appear meticulously crafted, the product of deep expertise, painstaking attention, master-level execution
- Leave creative space — specific enough for direction, concise enough for interpretive choices

### Philosophy Examples

**"Concrete Poetry"** — Communication through monumental form and bold geometry. Massive color blocks, sculptural typography, Brutalist spatial divisions. Ideas expressed through visual weight and spatial tension, not explanation.

**"Chromatic Language"** — Color as the primary information system. Geometric precision where color zones create meaning. Think Josef Albers meets data visualization. Information encoded spatially and chromatically.

**"Analog Meditation"** — Quiet visual contemplation through texture and breathing room. Paper grain, ink bleeds, vast negative space. Japanese photobook aesthetic.

## Step 2: Express on Canvas

With both the philosophy and conceptual framework established, express it visually:
- Use the design philosophy as the foundation
- Create one single page, highly visual, design-forward output
- Generally use repeating patterns and perfect shapes
- Treat the abstract philosophical design as if it were a scientific bible
- Add sparse, clinical typography
- Use a limited color palette that feels intentional and cohesive
- Text is always minimal and visual-first

**CRITICAL**: Create work that looks like it took countless hours. Make it appear as though someone at the absolute top of their field labored over every detail. Ensure composition, spacing, color choices, typography — everything screams expert-level craftsmanship.

### Key Facts

1. **Independent authorship:** Alexander's playbooks (`ui_playbook.md`,
   `integrated_product_building_workflow.md`, `unique_business_playbook.md`, etc.)
   were authored independently and predate the installation of these skill files.
   Any conceptual overlap (e.g., both discussing typography or color theory)
   reflects common domain knowledge, not derivation.

2. **No license contamination:** The MIT license on skill files does NOT extend to,
   dilute, or modify the CC BY-NC-SA 4.0 license on Alexander's original work in `docs/`.
   These are separate works with separate licenses.

3. **Direction of reference:** The skill files reference Alexander's playbooks as
   authoritative sources. Alexander's playbooks reference skills as optional
   supplementary tools. The playbooks are the ground truth; the skills are helpers.

4. **No merger:** These skill files are installed as standalone, isolated utilities.
   They are not merged into, woven into, or co-authored with the CC-licensed documents.
   The "Agent Skills Reference" sections in the playbooks are cross-references, not
   incorporation by reference.

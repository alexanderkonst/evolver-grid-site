# Growth Recipe Screen

## Goal
Visual screen showing "To grow in [QoL Domain], develop these 5 paths in this order" with ranked list.

## Context
From Crystalline Clarity Framework — all 5 Growth Paths contribute to every QoL goal, just ranked differently.

## Display
After user selects priority QoL domain, show:

1. Selected domain (e.g., "Wealth")
2. All 5 Growth Paths ranked by relevance:
   - 1. Uniqueness (primary)
   - 2. Mind 
   - 3. Spirit
   - 4. Emotions
   - 5. Body

3. Brief explanation for each path's contribution

## Example Rankings

### Wealth
1. Uniqueness — monetize genius
2. Mind — money beliefs, business thinking
3. Spirit — service orientation
4. Emotions — shadow work around money
5. Body — energy to execute

### Health
1. Body — physical foundation
2. Spirit — meaning and purpose
3. Emotions — stress, emotional eating
4. Mind — health beliefs
5. Uniqueness — authentic expression

## Location
Shown after QoL prioritization, before Daily Loop

## Acceptance Criteria
1. Shows 5 paths ranked for selected domain
2. Brief description for each path
3. Visual hierarchy (1st path highlighted)

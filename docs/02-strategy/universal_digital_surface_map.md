## Surface Physics Schema

### Persistence

| Level | Definition | Examples |
|-------|-----------|----------|
| **Ephemeral** | Designed to expire (~24h) | Stories, Status, WhatsApp Status |
| **Semi-persistent** | Remains but decays via ranking/feeds (algorithmic half-life) | Feed posts, comments, community posts |
| **Permanent** | Remains until changed/removed | Profile fields, pages, listings, libraries |

### Reach Type

| Type | Definition | Examples |
|------|-----------|----------|
| **Push** | Delivered to specific users via notification/email/inbox | DMs, email, notifications |
| **Broadcast** | Distributed via feed/recommendation/search ranking | Feed posts, YouTube videos, SEO pages |
| **Passive** | Discovered when someone visits a profile/page/listing | Bio, About section, banner |
| **Viral** | Explicitly designed to be reshared/forwarded/embedded | Reels, Shorts, shareable links |

### Effort Level

| Level | Definition |
|-------|-----------|
| **One-time setup** | Set and forget (profile, listing, headers) |
| **Periodic update** | Occasional maintenance (offers, schedules, pinned content) |
| **Ongoing creation** | Repeated publishing/engagement (posts, comments, episodes) |

### Paid Marker

| Level | Definition |
|-------|-----------|
| **No** | Free surface |
| **Optional** | Boost/promo available |
| **Yes** | Ad slot / sponsored placement; pay-to-play by design |

### Coach-Fit Rating

| Rating | Meaning |
|--------|---------|
| **Very High** | Core surface for service businesses — high trust + discoverability + conversion |
| **High** | Strong fit — should be activated |
| **Med** | Useful but not primary |
| **Low** | Marginal for this business type |

## High-Leverage Patterns for Service Businesses

> Surfaces that routinely outperform for coaches/consultants:

| Pattern | Why | Examples |
|---------|-----|---------|
| **High-trust passive surfaces** | Prospects see them late in decision cycle — "conversion accelerators" | Service Page reviews, Google Business reviews, testimonials |
| **Push surfaces that feel opted-in** | Repeated touchpoints without algorithm dependence | Newsletter subs, WhatsApp Channels, Telegram channels |
| **Search-and-maps surfaces** | Intent-rich contexts — attention is purchase-ready | Google Business posts/Q&A, SEO pages |
| **Link surfaces that collapse friction** | Single-lane funnel: attention → scheduled conversation | Booking link + intake questions + confirmation redirect |
| **Hybrid content surfaces** | Multiple entry points on one platform | YouTube community posts + Shorts + video + Live |

## The 5 Outputs of Every Session

One facilitator session produces five things simultaneously:

1. **A service delivered** — the founder gets their unique business forged
2. **A case study generated** — proof that the methodology works for this type of founder
3. **A playbook refinement** — new principles, sharper template, better session flow
4. **Alexander's own precision deepened** — the facilitator's articulation compounds
5. **A future platform user onboarded** — they'll use the Canvas + AI interface when built

This is the compound interest engine. One activity, five outputs. Each session makes all future sessions better.

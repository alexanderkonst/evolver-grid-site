# Artifact Identity 10x SOW

## Status

Next-phase scope of work for turning the Prime Radiant canonical artifact from a coherent internal object into a visually inevitable identity system.

This SOW does not replace `canonical_artifact_sow.md`. That document protects the source artifact. This document puts the artifact under real-world identity pressure.

Current phase: Phase 4 Derivation System is in progress. Evidence: `docs/assets/canonical-artifact/identity_derivation_notes.md`.

## Core Diagnosis

The artifact is now coherent, auditable, and ontologically grounded. That is not enough.

The current risk is that the system remains impressive as an internal metaphysical artifact but weak as a public identity. The next phase must make it survive practical identity constraints: small sizes, black/white usage, icon crops, deck covers, website headers, motion stings, and comparison against adjacent visual categories.

## 10x Thesis

10x is not more metaphysics. It is less explanation and more inevitability.

The artifact must work before the viewer reads the cosmology.

## North Star

Create a canonical identity system where one source artifact generates a mark, motion behavior, application rules, and presentation assets that feel inevitable across real contexts.

## Success Standard

The system is successful when a cold viewer can understand that the mark is precise, distinctive, intelligent, and alive without needing the full ontology explained first.

## Governing Constraints

1. The canonical model remains the source.
2. The 2D mark remains a projection or principled reduction, not a separately invented logo.
3. No decorative starbursts, particles, extra sacred-geometry overlays, or visual filler.
4. The artifact must not drift into generic luxury wellness, crypto geometry, AI-orb branding, or lab-tech cliché.
5. Public-facing copy must be radically compressed.
6. Identity applications must prove the system, not decorate a case study.

## In Scope

- Canonical hero selection.
- Hero render art direction.
- Material sharpening away from ordinary gold.
- Identity stress tests across real sizes and contexts.
- Projection / simplified mark derivation rules.
- Motion sting direction.
- Public-facing artifact sheet.
- Drift comparison against adjacent categories.
- Final recommendation: identity spine vs internal myth object.

## Out of Scope

- Changing the canonical octahedron/sphere/source geometry.
- Building a full brandbook.
- Rewriting the company strategy.
- Publishing assets publicly.
- Paid external rendering or fabrication unless explicitly approved later.
- Expanding ontology language instead of compressing it.

## Recursive Self-Check Protocol

After each phase, run a private-but-recorded check. Revise this SOW if evidence shows the plan is wrong.

Each check answers:

1. What did the last output reveal?
2. Which assumption was wrong or too vague?
3. Does the SOW or any DoD need revision?
4. Is the artifact becoming more visually inevitable or merely more documented?
5. Is there a decision that truly requires Sasha?
6. What is the one next irreversible action?

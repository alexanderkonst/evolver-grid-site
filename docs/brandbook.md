# Part II: Implementation â€” How to Apply

## Step 1: Context Check

Before designing, ask:
- Is this a **hero moment** (reveal, completion, celebration)?
- Is this **utility** (navigation, forms, containers)?
- Is this **dark mode** (depth, transformation) or **light mode** (action, business)?

---

## Step 2: Apply the Palette

| Context | Colors to Use |
|---------|---------------|
| Hero moments | Iridescent gradients (Strip 1 or 2) |
| Utility | Slate palette from `design_system.md` |
| Text on light | Charcoal Indigo (`#2c3150`) |
| Highlights & CTAs | Electric Violet (`#8460ea`) |
| Calm backgrounds | Pearl/Icy White (`#e7e9e5`) |

---

## Step 3: Check the Signal

After designing, ask:

> "Does every element carry signal, or is there noise?"

**If noise:** Remove it.
**If signal:** Amplify it.

## Step 4: Typography Consistency

| Element | Font Treatment |
|---------|----------------|
| Archetype names, hero text | Consider serif (gravitas) |
| Sacred quotes | Monospace (precision) |
| Body, navigation, forms | Sans-serif (utility) |

## Step 5: Motion Audit

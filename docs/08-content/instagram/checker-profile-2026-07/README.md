# Instagram checker-profile asset system

Production masters for a profile built for the checker, not the scroller.

## Output

- `output/feed/` — 4:5, 1080×1350 pinned carousels and covers.
- `output/story/` — 9:16, 1080×1920 Highlight sequences.
- `output/highlight-covers/` — 1:1, 1080×1080 cover art.
- `output/mockup/` — checker-view profile preview.

## Art direction

Contemporary East-Asian editorial restraint: midnight navy silk, architectural negative space, precise hairline gold, mineral-blue light, and one symbolic gesture per frame. Cormorant carries the mythic voice; Source Serif carries evidence and action. The system feels ceremonial without becoming occult, and premium without becoming decorative.

## Rebuild

```bash
node docs/08-content/instagram/checker-profile-2026-07/render.mjs
```

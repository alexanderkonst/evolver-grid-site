## THE SYSTEM (not a funnel — a field with intelligent entry points)

```
        (A) Warm Entry (PRIMARY — your network)
        ZoG → [gap felt] → Video → Session → Build

        (B) Cold Entry (social posts, shares)
        Quiz → ZoG → Video → Session → Build

        (C) Analytical Mind (overthinkers)
        ZoG → Quiz → Video → Session → Build

        (D) High Intent (instant recognizers)
        ZoG → Session (skip video)
```

> **Key insight:** The quiz is NOT the front door. The ZoG reveal IS.
> The quiz is "the hallway for people who need to look around before entering."

| Step | Job | User feels |

## 4 BUYER TYPES (behavior-based routing)

### Type 1 — Instant Recognizer
- **Behavior:** Takes ZoG, feels "holy sh*t", doesn't overthink
- **Show them:** "If this already makes sense — you don't need more thinking."
- **Route:** → Session immediately

### Type 2 — Sense-Maker
- **Behavior:** Takes ZoG, thinks "this is accurate… but why is this so hard?"
- **Show them:** "This explains why something that feels obvious hasn't turned into something real"
- **Route:** → Quiz → Video → Session

### Type 3 — Skeptic
- **Behavior:** Doesn't trust instantly, needs validation, prefers structure
- **Route:** Quiz first → ZoG → Video → Session

### Type 4 — Overthinker
- **Behavior:** Takes ZoG, loops mentally, doesn't act
- **Show them:** "You don't need more clarity. You need this finished."
- **Route:** → Direct to session

## ZoG REVEAL OUTPUT STRUCTURE (6-part format)

> The AI-generated result must follow this emotional arc exactly.

### 1. 🔮 THE NAME (Identity Anchor)
"You are: The [Pattern Name]"
- Keep it clean, not poetic-overload

### 2. 🪞 THE MIRROR (most important)
- Should feel like you've been watching their life
- Must feel: specific, undeniable, slightly uncomfortable
- Example: "You don't fit clean categories. You see patterns others miss—and instinctively reorganize things so they make sense."

### 3. ⚠️ THE DISTORTION (gap creator — where money is made)
- "The problem isn't your ability. The problem is: you've never turned this into a structure others can recognize and pay for."
- Creates tension + truth

### 4. 🧩 THE MISSED LINK
- "You're already doing the thing. You just can't see it from the outside."

### 5. 🚧 THE LIMITATION (STOP before resolution)
- "Because insight alone doesn't convert into income. Structure does."
- **DO NOT SOLVE THIS**

### 6. 🔥 THE BRIDGE
- "If this feels accurate — you're not missing clarity. You're missing the final step."
- → CTA: Watch video

### Step 1 — They engage ("this is me", "wow accurate")
❌ Don't: explain, pitch, send long messages

✅ Say: `You can see it… but you can't fully turn it into something real yet.`

✅ Say: `If you want, I can help you finish it in one session.`

✅ Say: `We take what you already do and turn it into a one-sentence business, a clear offer, and your next move—live.` → link

✅ Say: `No rush. People usually know when they're ready.`

## QUIZ ROLE (clarified)

> The quiz is NOT a lead magnet. It's a **controlled incomplete transformation.**

### For warm audience:
- Lead with ZoG reveal
- Quiz exists as optional deepener: "If this felt accurate but incomplete — this will show you why."

### For cold audience:
- Quiz IS the front door
- Frame: "Why is it still so hard to explain what you do?"

### What they walk away with:
- A named pattern (their type)
- A clear articulation of their problem
- A realization that this alone isn't enough

### What they should FEEL:
- "This is me… but I still can't turn this into something real."

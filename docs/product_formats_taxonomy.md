# Product Formats Taxonomy
## A Holonic Classification of How Value Gets Packaged

> *"The same genius can take infinite forms. The format is just the container."*

---

## Overview

Every genius-based offering can be packaged in different formats. This taxonomy helps creators choose the right container for their value.

---

## Primary Dimensions

### Dimension 1: Medium

| Medium | Description | Examples |
|--------|-------------|----------|
| **Digital** | Exists only in digital form | Courses, SaaS, ebooks |
| **Physical** | Tangible, material form | Books, retreats, workshops |
| **Hybrid** | Combination of both | Online course + in-person retreat |

### Dimension 2: Interaction Mode

| Mode | Description | Examples |
|------|-------------|----------|
| **1:1** | One provider, one client | Coaching, consulting |
| **Group** | One provider, multiple clients | Workshops, cohorts |
| **Self-paced** | No live interaction | Courses, templates |
| **Community** | Ongoing group container | Memberships, circles |

### Dimension 3: Time Structure

| Structure | Description | Examples |
|-----------|-------------|----------|
| **One-time** | Single transaction | Session, workshop |
| **Time-bound** | Fixed duration program | 8-week cohort |
| **Ongoing** | Continuous relationship | Membership, retainer |
| **Evergreen** | Available anytime | Course, template |

## Format Selection Matrix

Choose based on your Genius Business:

| If you want... | Consider... |
|----------------|-------------|
| Maximum scalability | Courses, Templates, SaaS |
| Highest price point | Retainers, Retreats, Masterminds |
| Lowest effort to start | 1:1 Sessions, Articles, Templates |
| Recurring revenue | Memberships, Retainers, SaaS |
| Community building | Cohorts, Memberships, Challenges |
| Passive income | Courses, Templates, SaaS |
